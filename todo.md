# TODO - AI Invitation Web App

Vue 3 + Tailwind CSS 3 | Elegant Blue Animated Invitation

## 🚀 Setup Project
- Inisialisasi project Vue 3
  - `npm create vue@latest`
- Masuk ke folder project
  - `cd your-project-name`
- Install dependencies
  - `npm install`
- Install Tailwind CSS 3
  - `npm install -D tailwindcss@3 postcss autoprefixer`
  - `npx tailwindcss init -p`
- Konfigurasi `tailwind.config.js`
- Tambahkan Tailwind ke CSS utama
- Jalankan project
  - `npm run dev`

## 🎨 Design System (Blue Elegant)
